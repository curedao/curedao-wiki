Kaiser Permanente Interchange API

Interchange enables internal and external developers to use approved public data from Kaiser Permanente to create individualized health care management apps. Using personalized health information, apps can be customized to cater to patient lifestyle, including nutrition and activity, to support total health. Interchange is a free API. The global rate limit is 5,000 transactions a day per each API Key. With Oauth, the global Interchange API rate is 2 transactions per second and 200 per hour, per token. When Oauth is not needed, the rate is 1,000 transactions per hour per API Key.
